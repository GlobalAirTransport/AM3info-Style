AM4 Tools new defined.

After almost 3 months of development, we are proud to announce the completion of a new and improved tool website for Xombat Airline Manager 4. We added a few new features along with very well known standard tools like the ticket price calculator or the seat configurator:

Guide Library
A place to collect useful guides for AM4. You can upload your guide here, please follow our guide upload rules.
Language Menu
English is not your mother tongue and you can barely understand it? Well, no problem. We decided to translate the website into several languages. If there's currently no translation available for your language, you can click here to add it!
Feature Suggestion/Bug Report
You have a cool new idea for a tool that might be useful? Or just an idea to improve the website?
We are always open for new ideas, please message us here.
Join the Community!
We have a well established Facebook Server and a Discord Server; all tools from the website are also available on Discord.
Get your own Discord Bot!
Since many of you also have private Discord servers, such as an alliance discord server, it is also possible to add the AM4info Bot to your own server.

Airline Manager 4

In Airline Manager 4, you start an airline from scratch. All you have to egin with is a few million dollars, and a single aircraft.
Do you have what it takes to run an airline?

Always wanted to run an airline?
Well, this is your change. But if you thought it was easy, think again. There are many tasks that must be done to become successful, and your staff are not much help - they also have to be managed.
Which aircraft will you purchase, and will your routes be profitable?
Sign up today and get started with your airline - it's free and 100% ad free.
